# Coffee-Sales-Dashboard
This project presents an interactive **Coffee Sales Dashboard** built in **Power BI**, designed to analyze sales performance across different coffee products, time periods, and customer behavior dimensions.
The objective of this dashboard is not just visualization, but to **answer stakeholder-driven business questions** and enable ongoing performance monitoring.
